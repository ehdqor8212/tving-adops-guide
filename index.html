Ad Ops 👋

티빙의 광고 운영을 책임지는 팀이에요.
청약 접수부터 캠페인 라이브, 모니터링까지 — 광고가 잘 돌아가도록 중심을 잡는 역할을 합니다.

CP / AP / NCA 팀이 열심히 세일즈해 주신 광고 물량의 운영 전반을 담당하는 팀입니다.

TAS(TVING Ad System)를 통해 청약 운영 및 관리를 진행합니다.

ABP에서 발생하는 모든 광고 매출과 비용에 대한 정산을 담당합니다.

이렇게 ABP 안에서 광고 집행의 시작부터 끝까지 함께하는 팀입니다.
📋

청약 관리

티빙광고팀으로 들어온 청약 집행에 이슈 없는지 판단 후 부킹 확정 안내를 하며, 파트너사에서 TAS 시스템에 청약을 세팅해주시면 검토하여 승인/반려합니다.

📡

캠페인 모니터링

집행 중인 캠페인이 정상적으로 노출되는지 별도 모니터링 시트를 통해 매일 확인하고, 이슈 발생 시 빠르게 대응합니다.

🔧

이슈 대응 · 시스템 개선

운영 중 이슈 발생 시 몰로코·PRODUCT팀과 함께 원인을 파악하고 해결합니다. TAS 개선 작업도 함께 진행해요.

🤝

부서 간 협의·조율

영업팀·GSE·PRODUCT팀 등과 상시 소통하며 광고 상품 운영 가능 여부, 신규 상품 검토, 개발 요청 등을 조율합니다.

🧾

광고 정산

광고 매출 비용에 대한 모든 정산을 담당합니다. 집행된 광고 금액을 기준으로 렙사·광고주와의 정산 전 과정을 처리합니다.

PROCESS

청약 프로세스

티빙 광고 집행을 위한 전체 프로세스입니다. 청약 요청부터 캠페인 라이브까지 4단계로 진행됩니다.

1
렙사 · 광고주

청약 요청 메일 발송

티빙 광고팀 공식 메일로 청약 요청 양식을 작성해 보냅니다.  청약 프로세스 →

  • 수신: tving.advertising@cj.net (담당자 개인 메일 접수 시 처리 불가)
  • 신규 광고계정인 경우 → 광고계정 생성 요청 양식 함께 첨부
  • 양식 종류: 예약형 광고 청약 요청 양식 · KBO 청약 요청 양식
  • 신규로 진행하는 렙사/대행사의 경우 계약서 작성 필요
2
티빙광고팀

낙찰 회신

영업일 기준 24시간 이내 티빙광고팀이 청약 확정 메일을 발송합니다. 메일에는 TAS 세팅에 필요한 상품 템플릿 정보가 포함됩니다.

  • 확정 메일 수신 후 TAS 접속하여 캠페인 세팅 진행 가능
  • 메일 내 템플릿 정보 = 청마파 기준 (이후 TAS 검토 시 대조 기준)
청약 요청 양식 예시
3
렙사 · 광고주

TAS 내 주문 생성

확정된 광고계정과 안내받은 템플릿 정보를 기반으로 TAS에서 직접 캠페인을 세팅하고 청약을 제출합니다.

  • 템플릿 선택 → 집행 기간 · 예산 · 소재 입력 → 청약 제출
  • 청약 하나 아래에 여러 개의 캠페인 생성 가능
  • 소재가 없으면 다음 단계(제출) 불가
4
티빙광고팀 (운영자)

청마파 대조 후 청약 및 소재 승인/반려

주문 제출 후 영업일 2일 이내 검수를 진행합니다. 청마파와 TAS 설정을 대조 후 승인하며, 시작일이 도래하면 광고가 라이브됩니다.

  • 일치 시 승인 · 불일치 시 반려 (사유 전달)
  • 심사 가이드 미준수 시 반려될 수 있음
  • 라이브 후 TAS 리포트에서 결과 데이터 확인 가능

→ 인바운드 메일 성격상 특정 담당자가 드러나지 않도록 하기 위함

TAS

TAS란?

TAS(TVING Ad System)는 몰로코에서 티빙의 광고 운영을 위해 개발해준 플랫폼이에요. 파트너사가 광고 청약을 직접 등록하고, 운영자가 마스터 계정에서 검토·관리하는 구조입니다.

⚙️

운영자 (마스터 계정)

청약 승인/반려 · 소재 검수 · 타겟·템플릿 생성 · 페이싱 관리

👥

파트너사 (렙사 계정)

청약 등록 · 소재 업로드 · 리포트 보기

운영자 계정 등록 방법
1 옵스 또는 소속팀을 통해 CJ 이메일을 전달해주세요. 운영자 계정으로 등록해 드립니다.
2 등록 완료 시 이메일로 TAS 계정 생성 안내가 발송돼요. 해당 메일로 TAS 가입을 완료하면 끝!
3 마스터 계정 가입이 완료되면 렙사 계정도 직접 등록할 수 있어요.
렙사 계정 등록 방법
1 TAS 메인 화면 우측 상단 이름 클릭 → 광고 계정 및 브랜드 관리 선택
2 광고 계정 유저 탭 선택 후 TVING (TVING Inhouse) 검색
3 유저 초대하기 버튼을 클릭하여 렙사용 계정 이메일 등록
⚠ 주의사항

렙사 계정용 이메일은 마스터 계정 이메일과 달라야 해요.

💡 TIP

렙사용 계정 이름은 김티빙_렙사 형식으로 구분해 두면 관리가 편해요!

마스터 계정 (운영자)

TAS 전체를 관리하는 계정이에요. 타겟·템플릿 생성부터 청약 검토, 집행 모니터링까지 운영 전반의 권한을 가집니다.

  • 타겟 생성 · 관리
  • 캠페인 템플릿 생성 · 관리
  • 청약 검토 · 승인 / 반려
  • 광고 소재 검수
  • AD Pacing — 소진 예산 확인 · 우선순위 조정
  • 전체 광고계정 주문 리스트 열람
  • 리포트 열람 (전체 계정)
렙사 계정 (파트너사)

광고주·렙사 측 담당자가 사용하는 계정이에요. 운영자가 만들어둔 타겟과 템플릿을 바탕으로 직접 청약을 생성·제출합니다.

  • 예약형 주문(청약) 생성 · 제출
  • 광고 소재 업로드
  • 담당자 초대 관리
  • 하위 브랜드 추가
  • 자기 계정 주문 리스트 열람
  • 리포트 열람 (자기 계정 데이터만)
운영자가 만든 타겟과 템플릿은 모든 광고계정에서 공용으로 사용 가능합니다.

타겟 생성부터 집행 관리까지, 누가 어느 단계를 담당하는지 확인해볼까요?

예시 상황
삼성전자 광고주가 베이직 인스트림 상품에 여성 2049 성·연령 타겟팅을 하여 15초·30초 캠페인을 운영하고 싶은 경우, 아래와 같은 단계로 진행돼요.
🟢
운영자

① 광고계정 생성 · 렙사 담당자 초대

마스터 계정에서 광고계정을 생성하고, 렙사 담당자를 해당 계정에 초대합니다. 담당자는 초대 이메일을 통해 TAS에 접속할 수 있어요.

🟢
운영자

② 타겟 생성

마스터 계정에서 타겟을 만듭니다. 모든 템플릿의 기반이 되는 단위예요.
타겟은 템플릿의 재료예요 — 동일한 타겟(여성 2049)이라도 15초·30초처럼 상품이 다르면 CPM 단가가 달라지는데, 단가는 타겟이 아닌 템플릿에서 설정하거든요.
예: 여성 2049, 남성 1834, 프리롤, 미드롤

🟢
운영자

③ 템플릿 생성

타겟 + 상품 유형 + 단가를 조합해 템플릿을 만듭니다. 모든 광고계정에서 공용으로 사용할 수 있어요.

🟢
운영자

④ 템플릿 정보 메일 안내

생성된 템플릿 정보(상품명·CPM 단가·타겟 등)를 티빙광고팀 메일로 렙사 담당자에게 회신합니다. 담당자는 이 내용을 기준으로 TAS에서 캠페인을 세팅해요.

렙사 담당자

⑤ 캠페인 생성

원하는 템플릿을 선택하고 아래 항목을 입력합니다. 하나의 설정이 곧 1개 캠페인입니다.

일정
시작일·종료일 (최대 90일), 타겟 일정 필수
예산
유가 청약 → 비용(₩)에 금액 입력
무가 청약 → 보너스율(% 또는 원)에 금액 입력
광고 소재
소재 생성 또는 기존 소재 추가 (필수)
옵션: 랜딩 URL · 광고 스케줄 · 주간 일정 · 소재별 가중치 설정 가능
🔵
렙사 담당자

⑥ 청약 생성 · 제출

청약이 캠페인의 상위 단위입니다. 청약 하나 아래에 여러 개의 캠페인을 생성할 수 있으며, 단일 캠페인으로만 구성하는 것도 가능합니다.

🟢
운영자

⑦ 청마파 대조 후 청약 및 소재 승인/반려

담당자가 모든 설정을 완료한 상태입니다. 운영자는 청마파(티빙광고팀 메일로 받은 청약폼 기준) 내역과 TAS 설정이 일치하는지 확인 후 승인 또는 반려합니다.
캠페인 승인 후 소재 교체·랜딩 변경 등 추가 수정이 필요한 경우, 렙사 담당자가 청약을 수정·재제출하면 운영자가 변경 내용을 확인하고 승인/반려합니다.

🟢
운영자

⑧ AD Pacing · 리포트

집행 중 캠페인별 소진 예산을 확인하고, 필요 시 우선순위를 조정합니다 (마스터 전용).
리포트는 운영자·담당자 모두 열람 가능합니다.

📌 참고

운영자·담당자가 볼 수 있는 칼럼 값에는 차이가 있어요.
예) 대표적으로 렙사용 TAS에는 채널 칼럼 값이 존재하지 않습니다.

실전 예시

삼성전자 갤럭시 S26 캠페인을 예시로 전체 구조를 살펴봅니다.

1단계 — 운영자: 타겟 생성
타겟명여성 2049
조건여성 · 만 20~49세
2단계 — 운영자: 템플릿 생성 (전체 광고계정 공용)
[통합]베이직인스트림_MF2049_논스킵_15초CPM 21,000원
[통합]베이직인스트림_MF2049_논스킵_30초CPM 42,000원
3단계 — 렙사 담당자: 캠페인 설정
브랜드갤럭시 S26
캠페인 115초 · 4/7~4/30 · 예산 500만원
캠페인 230초 · 4/7~4/20 · 예산 300만원
청약캠페인 2개를 묶어 청약 1건으로 제출
4단계 — 운영자: 청마파 대조 후 청약 및 소재 승인/반려
확인 항목청마파(티빙광고팀 메일 청약폼) ↔ TAS 설정 일치 여부
결과일치 시 승인 · 불일치 시 반려(사유 전달)
